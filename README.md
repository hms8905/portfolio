# 🧑‍💻 Frontend Portfolio

프론트엔드 개발자 함명식의 포트폴리오입니다.
React + TypeScript 기반으로 사용자 경험과 유지보수성을 고려하여 제작했습니다.

---

## 🚀 Live Demo

👉 https://hms8905.github.io/portfolio/

---

## 🛠 Tech Stack

* React
* TypeScript
* Vite
* SCSS (module)
* GSAP
* Swiper

---

## ✨ 주요 특징

### 1. 구조 설계

* sections / components 분리
* Layout 기반 구조
* 유지보수를 고려한 폴더 구조

### 2. 인터랙션

* GSAP 기반 scroll animation
* Hero / About / Projects 인터랙션 분리
* 사용자 경험을 해치지 않는 수준의 모션 설계

### 3. 반응형 & 접근성

* rem 기반 디자인
* prefers-reduced-motion 대응
* focus-visible 적용

### 4. 성능 고려

* 이미지 lazy loading
* Swiper 최적화
* 불필요한 animation 최소화
